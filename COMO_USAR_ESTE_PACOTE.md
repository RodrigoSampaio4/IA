# Como usar este pacote

Este `.zip` contém a estrutura final do repositório da Etapa 4 do projeto AgTech.

Para enviar ao GitHub:

1. Extraia o arquivo `.zip` no computador.
2. Abra a pasta `AgTech_Automacao_Precisao_Etapa4_Final`.
3. Envie para o GitHub os arquivos e pastas internos:
   - `README.md`
   - `requirements.txt`
   - `notebooks/`
   - `data/`
   - `src/`
   - `logs/`
   - `assets/images/`
   - `docs/`
4. Não envie o arquivo `.zip` para o repositório, pois a Etapa 4 pede os arquivos expostos em pastas.
5. Grave o vídeo de demonstração, suba no YouTube ou Google Drive público e substitua `LINK_DO_VIDEO` no `README.md`.
